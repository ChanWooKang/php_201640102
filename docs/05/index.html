수업 내용 정리


오늘은 PHP에서 다양한 함수와 클래스들을 배웠다.


Include활용

Include란 다른 파일을 하나에 파일에 불러오는 역활을한다



Include 코드를 쓴만큼 같은 파일을 똑같이 불러올수도있다.



만약 존재하지 않은 파일을 가져온다고 코딩했을경우 오류가 뜬다.



이와 같은 경우에선 Require 함수를 사용한다 (만약 없을경우 실행이 중단)



Require도 마찬가지로 중복으로 사용이 가능하다.




Function




Array 배열은 변수로 받거나 []으로도 만들수 있다





배열은 include로도 받을수 있다





function_exsist(함수명)를 사용하면 괄호안에 있는 값을 이름으로 하는 함수가 있는지 확인한다
있으면 true 없으면 false로 받는다 그러므로 if문으로 사용가능하다



Class

include Class로 받기



함수는 선언후 바로 호출인 2단계 구조인 반면 클래스는 선언 생성 호출 3단계 구조이다 이때 생성되는동작을 인스턴스화라고 한다.






const(상수) 상수란 변하지 않은 수이며 값을 지정해둔다.



this 와 self에 공통점은 둘다 클래스자체를 말한느것이고 차이점은 클래스가 생성이 되었는지 안되었는지(인스턴스화) 차이점이다




메소드 체인 이름 그대로 메소드를 따로따로 호출하는것이아닌 한줄로 호출하는 방식이다





static 을 사용하면 클래스 생성을 따로 하지 않아도 된다.




그러므로 this 가 아닌 self 를 사용한다.



클래스 정리하자면 함수와 다른 3단계 구조이고 클래스안에 있는 함수는 메소드 바깥에 있는것을 함수라고 칭한다 3단계 구조 중 생성을 인스턴스화라고하며 생성된 클래스를 객체,오브젝트,인스턴스라고 한다.






construct는 자동으로 함수를 생성한다 __CLASS__ = 자기자신(클래스)이름
