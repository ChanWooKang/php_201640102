수업 2019. 10. 02

수업 내용 정리


오늘은 수업시간에 중간고사 예제문제를 보느라 많은것을 나가진 않았다.



세션함수를 이용하여 처음화면에서 저장시키는 값을 사용하는 것인데
문제는 제 노트북에서 권한 오류가 계속 떠 적용이 안되었다.



폴더안에있는 모든 파일 및 폴더를 출력한다.



폴더안에있는 하위폴더를 파일과 구분하기위해 글씨체를 굵거 한다..



문자열이 컴퓨터상으로 어떻게 코딩이 되는지 확인하는 함수


오늘은 수업진도보단 중간고사 초점에 맞춰서 수업해주셨습니다.
